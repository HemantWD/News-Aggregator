News API

News Appgregator API

  • METHOD: POST
    Description: Post request to fetch API data and store in the database.

    API: http://localhost:5500/api/news

  • METHOD: POST
    Description: POST request to create a single article.

    API: http://localhost:5500/api/new-news

  • METHOD: GET
    Description: Get request for getting a single article.

    API: http://localhost:5500/api/news/:id

  • METHOD: PUT
    Description: PUT request to update a news article.

    API: http://localhost:5500/api/news/:id

  • METHOD: DELETE
    Description: Delete request for deleting an article.

    API: http://localhost:5500/api/news/"id

  • METHOD: GET
    Description: Request for filtering news based on category.

    API: http://localhost:5500/api/filter

  • METHOD: GET
    Description: Get request for searching with a keyword.

    API: http://localhost:5500/api/search
